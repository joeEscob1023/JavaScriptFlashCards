JavaScript Functions Click here

A "JavaScript Function" is a set of repeatable code that runs when it is called back. It can be used for almost anything and have multiple statements to make the program do an action.

(EX:)
function saySomething(word) {
return word;
}
saySomething(prompt('What Do You Want The Computer To Say?'));

This calls a function called saySomething and it returns what ever you put into the prompt.

Try this in your console and see how it works. Just copy and paste it in and press enter.

JavaScript jQuery Library Click here

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. With a combination of versatility and extensibility, jQuery has changed the way that millions of people write JavaScript. In other words, you can write more JavaScript without writing a ton of code.
